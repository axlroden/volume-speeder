Volume Speeder

Speed up your volume keys: repeat each press by N. Also works great for volume wheels that you feel are too slow, like the Lofree Keyboards. Elegant tray UI on Windows/macOS. Lightweight CLI on Linux.

Latest Release Install GitHub

Windows

Wails GUI with tray icon, modern flyout, configurable multiplier.

One-liner (user install):
powershell -ExecutionPolicy Bypass -NoProfile -Command "iwr -useb https://raw.githubusercontent.com/axlroden/volume-speeder/HEAD/install.ps1 | iex"
NSIS silent:
VolumeSpeederSetup.exe /VERYSILENT /NORESTART

macOS

Wails GUI app bundle. Unsigned (no Apple Developer ID), so Gatekeeper may warn.

One-liner (auto-trust):
sh -c "TRUST=1 $(curl -fsSL https://raw.githubusercontent.com/axlroden/volume-speeder/HEAD/install.sh)"
Or run without trust, then open manually:
sh -c "$(curl -fsSL https://raw.githubusercontent.com/axlroden/volume-speeder/HEAD/install.sh)"
# Then: right-click Volume Speeder in /Applications → Open → Open

Linux

Tiny CLI with config at ~/.config/volume-speeder/config.

One-liner:
sh -c "$(curl -fsSL https://raw.githubusercontent.com/axlroden/volume-speeder/HEAD/install.sh)"
Examples:
volume-speeder get
volume-speeder set 5
volume-speeder inc

Install

Prefer the one-liners above, or download artifacts from the release page:

  • Windows: .exe and NSIS setup
  • macOS: .dmg (or .zip)
  • Linux: .tar.gz (amd64)

How it works

Windows/macOS provide a tray app that multiplies volume key presses. Linux keeps it simple: a CLI that stores your multiplier in ~/.config/volume-speeder/config.
